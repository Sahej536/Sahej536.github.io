CSS Framework

Bootstrap

When you are building any website or webpage then frameworks work as supporting structures that have already been developed and are used to create any web application. It’s basically reusable code that provides specific functionality in your programs. It helps you to create the website design faster and easier because you don’t have to rewrite the code but if you want to make any change you can do that. We will talk about the CSS Frameworks- Bootstrap. Bootstrap is one of the popular HTML, CSS, and Javascript frameworks for developing mobile-friendly websites.

Here are some points that tells why to use bootstrap-

  • It is an open-source platform.
  • Excellent Browser Compatibility.
  • Easy to use.
  • Mobile-First Approach.
  • Responsive Features.
